
# Deployment Instructions

## GitHub Push
1. Copy all files to your local repo folder.
2. Run:
   git add -A
   git commit -m "Add all layers and update map"
   git push origin main

## Embed in Cargo Collective
Use the following iframe code:

<iframe src="https://yourgithubusername.github.io/yourrepo/index.html" width="100%" height="600" frameborder="0"></iframe>
